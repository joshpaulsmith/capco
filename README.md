# Capco Construction

Static website for Capco Construction in Sault Ste. Marie, Ontario.

## Structure

- `index.html` - homepage
- `portfolio/index.html` - project gallery
- `styles.css` - site styles
- `script.js` - mobile nav, reveal animations, and portfolio filtering
- `assets/` - optimized project photos used by the site

## Deployment

This repository is set up as a static site and can be deployed directly with GitHub Pages or any static host.

## Updating content

- Homepage copy and SEO metadata live in `index.html`
- Portfolio copy lives in `portfolio/index.html`
- Portfolio items are managed in `script.js`
- Project images should be added to `assets/` as web-friendly `.jpeg` files

Avoid committing `.heic` originals to the deploy branch. Convert and optimize photos before adding them to the site.
